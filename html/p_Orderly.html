This project is a large official website design project, I also need to collaborate with UX designers to complete this work.

The visual style of this website uses the original illustration style of Orderly system to extend the design, like icons, color scheme, characters, etc., are all designed according to the original guideline, but I also make new changes in this project, so that the new website illustrations are more lively and lovely, and make the website more exquisite.

The semi-transparent "o" pattern in the background of the website not only symbol the "o" in the name of Orderly, but also symbol the shape of the illustration character in the website, increasing the connection between the website and the brand.
