This project actually is a concept website. The main idea is to change the website to be as trendy and cool as the brand style, and keep the humanized operation.

I used FILA's logo color to match the website color. When on the product page, the screen is divied into left&right part. So users could view the product clearly.

All of the main design elements are composed of sharp squares and rectangles to creat a neat and clean visual style.
