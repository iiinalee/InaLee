"Period Calendar" is an application that can predict women's menstrual period and ovulation day, and can record various physiological symptoms, just like girls' portable secretaries.

I have used this app for a long time, and I like it very much because it's very convenience.

But I also found that there are some shortcomings in this app, such as functional optimization, visual design, etc., so I made a redesign of this project.

01. Homepage

Simplify the information on the homepage, and highlight the main function of "tracking menstruation", and make the visual style to be more clean and gentle.

In addition, according to the arrival of menstruation, the color of the interface will change accordingly.

02. Record

Unify the visual style of the icon on the record page, and change the original column-based options to a chart in the nine-square format, make icons instead of text to convey option information.

03. Report

Let users to view the body report information more directly, they don't have to click the classification to view it. Users only need to drag the icon below to view others status reports.

Moreover, I put the key report information on the top of the interface, highlight the information, and they can click the "details" button to view more information.
